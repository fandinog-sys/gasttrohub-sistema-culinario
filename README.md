\# GastroHub - Plataforma de Gestión Culinaria



\## Descripción

GastroHub es una plataforma colaborativa diseñada para la gestión, descubrimiento y planificación de recetas. El sistema permite optimizar la cocina diaria basándose en presupuestos, preferencias dietéticas e ingredientes disponibles.



\## Alcance

El proyecto abarca desde la creación de recetas profesionales por chefs hasta la personalización de versiones y planificación de menús semanales para usuarios regulares.



\## Convención de Commits

Para mantener el orden en el desarrollo, utilizaremos los siguientes prefijos:

\* \*\*feat:\*\* Nueva funcionalidad.

\* \*\*fix:\*\* Corrección de errores (bugs).

\* \*\*style:\*\* Cambios de formato, CSS o diseño (sin cambiar lógica).

\* \*\*docs:\*\* Cambios solo en la documentación (.md).

\* \*\*refactor:\*\* Mejora de código que no añade funciones ni arregla fallos.
